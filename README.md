## Usage

To use the library you need to have follwing things added to build gradle

```
repositories {
    jcenter()
    maven { url "https://jitpack.io" }
}

dependencies {
    implementation 'com.github.ubikampus:ubi-Indoor-Positioning:v0.1.0'
}
```
